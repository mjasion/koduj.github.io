    Code Club World to światowa sieć darmowych, pozalekcyjnych klubów kodowania przeznaczonych dla dzieci w przedziale wiekowym 9-11 lat, które prowadzone są przez wolontariuszy. Projekt ten wywodzi się z brytyjskiej inicjatywy Code Club UK.

  • Czym jest Code Club World?

    Code Club World to światowa sieć darmowych, pozalekcyjnych klubów kodowania przeznaczonych dla dzieci w przedziale wiekowym 9-11 lat, które prowadzone są przez wolontariuszy. Ma też swój polski odpowiednik Code Club Polska i pierwsze trzy działające kluby.

    Organizacja tworzy projekty dla swoich wolontariuszy, tak aby mogli oni uczyć w ramach darmowych pozalekcyjnych Klubów Kodowania. Projekty te uczą dzieci programowania pokazując im jak tworzyć gry komputerowe, animacje czy strony internetowe.

    Wolontariusze prowadzą raz w tygodniu jednogodzinne zajęcia w szkołach podstawowych lub innych miejscach takich jak biblioteki, świetlice, domy kultury. Na każdym spotkaniu dzieci wykonują jeden projekt. W trakcie I i II semestru, uczniowie używają Scratch by opanować podstawy zasad programowania. Semestr III uczy podstaw tworzenia portali internetowych przy użyciu HTML oraz CSS. Semestr IV uczy programowania w Pythonie i tak dalej.

  • Znalezienie odpowiedniego miejsca

    Jeśli jesteś ochotnikiem i chcesz prowadzić Klub Kodowania będziesz musiał znaleźć odpowiednie miejsce. Koniecznym jest by wybrane miejsce zapewniało dostęp do komputerów.

    Biblioteki oraz świetlice również mogą stanowić doskonałe miejsca na przeprowadzanie zajęć klubowych w weekendy czy poza godzinami lekcyjnymi.

    Ważne jest, by bezpośrednio skontaktować się z placówką i zaaranżować spotkanie twarzą w twarz. Konieczne jest wyjaśnienie czym jest Code Club i zaprezentowanie przyniesionych ze sobą odpowiednich materiałów.

    Potem można tylko zaproponować terminy odbywania się zajęć i organizacja klubu jest gotowa! Ostatnim krokiem przed pierwszym spotkaniem będzie rejestracja swojego klubu na stronie Code Club Polska oraz pobranie materiałów z Github.

  • Prowadzenie klubu

    Każde spotkanie Code Club’u trwa 1 godzinę i odbywa się raz w tygodniu.

    Za prowadzone zajęcia w ramach klubu nie wolno pobierać opłat. Code Club jest darmowy dla wszystkich.

    Prowadzenie dobrego Code Clubu może zostać osiągnięte dzięki ustaleniu regularności. Jeśli dzieci wiedzą, co mają robić oraz czego mogą się spodziewać to jest łatwiej je uczyć, a im będzie łatwiej przyswoić wiedzę.

    Wszystkie materiały dydaktyczne są dostępne dla każdego na portalu Github (link) w formacie markdown. W ramach Code Club Polska wolontariusze tłumaczą materiały dydaktyczne z wersji angielskiej. Polskie materiały dydaktyczne w formie, którą dzieci otrzymują na zajęciach można zobaczyć na stronie projects.codeclubworld.org .

  • Znalezienie wolontariuszy

    Wolontariat w klubach kodowania polega na przybyciu raz w tygodniu do miejscowej szkoły podstawowej lub innego miejsca w celu przeprowadzenia godzinnych zajęć klubowych. Wielu wolontariuszy prowadzi kluby wspólnie z kolegami lub przyjaciółmi - pomagają sobie wtedy nawzajem lub prowadzą zajęcia na zmianę.

    Jako prowadzący zajęcia w ramach Code Club każdy ma dostęp do powstałych projektów (link) , które służą jako materiały do pracy z dziećmi w szkole. W projektach używa się Scratcha przez pierwsze dwa semestry, aby nauczyć dzieci podstawowych elementów programowania. Następnie przechodzi się do języków HTML i CSS oraz Python.

    Code Club został tak zaprojektowany, aby przez zabawę pobudzać kreatywność, a dzieci dobrze się bawiły w czasie spędzonym w Klubie Kodowania, który nie wygląda jak kolejna zwykła lekcja. Powinny zrozumieć, że to one panują nad komputerem i że komputer może (a nawet powinien) robić to, co one każą mu wykonać, a nie odwrotnie.
